## 📖 简介

AstraFlow AI 是一个面向企业业务场景的智能工作台，融合 Agentic RAG、知识库问答、工具调用和 AI 自动化流程编排，帮助用户完成从知识查询到任务执行的智能化闭环。由FastAPI-Vue-Admin 基础平台二次开发，

以下改动；
1.sqlite 改为 mysql 数据库
2.Tortoise ORM 改为 SQLAlchemy
3.增加 Agentic Rag 工作流
